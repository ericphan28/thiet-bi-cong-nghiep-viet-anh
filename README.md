# Bánh xe đẩy HCM

Website giới thiệu sản phẩm và dịch vụ cho công ty chuyên cung cấp bánh xe đẩy tại TP.HCM.

## 🚀 Tính năng

- ✅ Hiển thị danh mục sản phẩm đa dạng
- ✅ Chi tiết sản phẩm với variants
- ✅ Responsive design 
- ✅ SEO optimized
- ✅ Placeholder images system
- ✅ Contact forms
- ✅ Company information

## 🛠️ Công nghệ sử dụng

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Radix UI** - Component library
- **Lucide React** - Icons

## 📁 Cấu trúc project

```
app/
├── page.tsx                 # Trang chủ
├── san-pham/               # Trang sản phẩm
│   ├── page.tsx            # Danh sách sản phẩm
│   └── [categoryId]/       # Sản phẩm theo danh mục
├── gioi-thieu/            # Trang giới thiệu
├── lien-he/               # Trang liên hệ
components/
├── layout/                # Header, Footer
├── sections/              # Các section trang chủ
├── ui/                    # UI components
└── ProductCard.tsx        # Card sản phẩm
lib/
├── mock-data.ts          # Dữ liệu mẫu
└── utils.ts              # Utilities
```

## 🚀 Chạy project

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm run dev

# Build production
npm run build

# Start production server
npm start
```
